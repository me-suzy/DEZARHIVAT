csCreate by CGIScript.net

Instructions:

  • Create a new page allows you to create a page from scratch.

  • Import an existing page lets you select from existing pages on the server.

  • Create a directory allows you to add directories to the server.

  • Edit allows you to modify the file live on the server.

  • Delete permanently removes the file from the server.

  • Remove simply eliminates the file from the csCreate list, the file remains on server.
