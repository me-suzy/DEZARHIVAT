CSFileManager can perform many of the file operations needed to manage a web site. You can use it to navigate the directories of your server, viewing a list of files in each. You can also copy, create, delete, edit, move and upload files, change file permissions, as well as create and delete subdirectories.

Index of the current directory: in(disdir)

Click on any name to view a file or move into a directory. Or select the button next to a file or directory and then choose from the list of actions below.

Action Instructions

Here is a list of actions and details of what the action does.
Upload a File Add a new file. Click 'Upload' button and select a file on your local hard drive.
Edit a File Edit an existing file. Select file from Index above and click the 'Edit' button.
Move a File Move a file or directory to a new location. Select a file or directory from the Index above then click the 'Move' button and type the new location in the popup window.
Rename a File Rename a file or directory. Select a file or directory from the Index above then click 'Rename' button and type the new name for the file/directory in the popup window.
Create a File Create and begin editing a new file. Click the 'Create' button to launch text editor.
Create a Directory Create a new directory. Click the 'Create' button and type the name of the new directory in the popup window.
Delete a File/Directory Delete an existing file or directory. Select a file or directory from the Index above then click 'Delete' button. Warning, if you delete a directory you will lose all of the contents as well.
Copy a File Copy an existing file to a new location. Select a file or directory from the Index above then click the 'Copy' button and type the new location and file name in the popup window.
CHMOD Change the file permissions of a file or directory. Select a file or directory from the Index above then click the 'CHMOD' button and set the new file permissions in the popup window.
