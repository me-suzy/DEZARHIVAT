E-mail Newsletter Readme: alpha build 1

No full readme yet; quick install instructions follow.

- Upload all files to your Coranto directory.
- Edit emailsub.cgi to reflect the appropriate path to perl.
- Chmod emailsub.cgi so that it's executable (755, generally)
- Edit emailsub.tmpl as necessary
- Enable addon via Addon Manager
- Create new E-mail Newsletter profile via Manage Profiles
- Enjoy
