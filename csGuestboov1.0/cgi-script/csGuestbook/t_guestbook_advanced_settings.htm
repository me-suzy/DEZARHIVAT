Guestbook Options
Approve Submissions
If this option is checked, new submissions will go to a holding area until you approve them. If you select this option, you should also enable email notification of new entries so you know when they have been posted.
Delete Entries After
Days If this option is selected, guestbook entries will be deleted after XX days since it was first submitted to the guestbook. To disable, enter '0'.
Message Limit 
Characters If this option is selected, guestbook entries will be limited to  XX charcters. To disable, enter '0'.
