Search Options
Note: If this is a new installation and you are using Indexing, which is the default method, you MUST click the Reindex Now button to create your index. Otherwise, a search will have no results.
Use Indexing
The index option creates an index of all the words on your website. This option is MUCH faster than non-index searching since it doesn't have to search through all your files each time a search is performed. When it does a search, it only needs to look at the index file and see what words are on what pages. It is also recommended that you use indexing if you have a large website. The only drawback is if you make any changes to your site, you need to go back and reindex so the new index will reflect the changes you have made. If you change any search options, such as, ignoring meta tags or parsing for server side includes.. you'll have to first save your changes and then come back to this advanced settings page and click the Reindex button for the search results to reflect the changes.
Ignore Title Meta Tag
check this box to ignore the <title></title> tag during searches
Ignore Keyword Meta Tag
check this box to ignore the keywords meta tag during searches. This is helpful when you repeat the same information in the keywords meta tag on multiple pages and you don't want the search engine to pick them up.
Ignore Description Meta Tag
check this box to ignore the description meta tag during searches. This is helpful when you repeat the same information in the description meta tag on multiple pages and you don't want the search engine to pick them up.
Parse Server Side Includes
note: this requires the LWP module to be installed on your server and will slow down the indexing (or searching if you have indexing disabled) speed drastically. It is recommended that you leave this feature unchecked unless you have server side include pages on your site that need to be searched.
File Extensions You can include or exclude file type in mass by typing the file extensions below. For example, if you want to ONLY search htm and html files then select "Include" and type htm, html in the text field. The search will ONLY search htm and html files. If you wish to search all files EXCEPT certain file extensions select "Exclude" and type the file extensions you wish to EXCLUDE. If you type gif, jpg, avi the search will EXCLUDE gif, jpg and avi files.
