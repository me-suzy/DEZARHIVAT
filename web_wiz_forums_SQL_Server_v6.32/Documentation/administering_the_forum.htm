Administering the Web Wiz Forums
Return to the main menu

  1. Once the your Web Wiz Forums is up and running you can administer and set up your forum by logging in through the forums login page with the default admin username and password is: -
    • Username: - Administrator
    • Password: - letmein
  2. Now you have logged into the forums admin account click on the admin link in the top left corner of the forum to take you to the administration and set up pages.
  3. In the admin section of the forum you can find the following:-
    • Set up new forums and categories - including security settings
    • Change the default admin username and password - highly recommend
    • Setup and turn on and off e-mail functions of the forum
    • Configure the forum - turn on and off forum functions and change the forum logo
    • Set forum moderators
    • Configure time and date settings - also change the time displayed by the forum to your local time
    • Anti-spam configuration
    • Configure the bad word filter
    • Archive old topics
    • Batch delete old topic
