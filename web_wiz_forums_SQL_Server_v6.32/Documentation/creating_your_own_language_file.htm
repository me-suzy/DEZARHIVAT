Writing Your Own Language File
Return to the main menu

If a language file is not available to translate the forum into your own language you can easily translate the forum your self into your own language. To do this simply follow the instructions below: -

  1. Make a back-up of the file langauge_file.inc.
  2. Open the file langauge_file.inc in a text editor like Notepad.
  3. In this file you will see lots of lines that look like the line below.

    Const strTxtWelcome = "Welcome"

  4. Translate the parts shown above in red between the quotation marks to your own language.
  5. Once you have finished altering the langauge_file.inc save the amended version and test your forum.
  6. You can also translate your forums sign up rules and policies by editing the file registration_polices_file.inc in Notepad or another text editor.
