Installing a new Language File
Return to the main menu

There are various different language files available for the Web Wiz Forums from Web Wiz Guide that allow you to change the language of your forum. Follow the instruction below to install and new language file for your forum.

  1. Unzip the language zip file onto your hard drive.
  2. Make a back-up of the file found in the forum files called langauge_file.inc in case things don't quite go to plan.
  3. Replace the langauge_file.inc found in the forum files with the language file you wish to install.
  4. That should be it, if there are any additional instructions for installing the language file you have download then check the ReadMe.txt file or any other documentation that comes with the language zip file.
