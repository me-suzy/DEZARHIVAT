Installing a New Skin
Return to the main menu

There are various different skins available for the Web Wiz Forums from Web Wiz Guide that allow you to change the look and feel of the forum. If you download one of these skins then follow the instructions below to install the new skin.

  1. Unzip the skin zip file onto your hard drive keeping the files in there original directories.
  2. Make a back-up of all your forum files in case things don't quite go to plan.
  3. Replace the skin_file.inc found in the forum files with the one for the new skin.
  4. If your new skin also comes with images in a directory called forum_images then overwrite the original images found in the forum_images folder with the new images.
  5. If your new skin has a file called smileys containing new emoticons images then overwrite the images found in the forums smileys directory with the new images.
  6. That should be it, if there are any additional instructions for installing the skin you have download then check the ReadMe.txt file or any other documentation that comes with the skin zip file.
