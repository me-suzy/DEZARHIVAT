Integrating the Forum Into Your Web Site
Return to the main menu

As well as changing the look and feel of the forum through the skin file you can also easily integrate the forum into your own site by editing three files to add your own sites logo across the top, left side bar navigation system, and any info you have in the footer of your site.

  1. Make a back-up of the files, header.inc, side_bar.inc, and footer.inc.
  2. Open the file header.inc in an HTML editor (it is strongly recommended that you do NOT use Frontpage) and add your own web sites logo or any other content you wish display at the top of each page of the forum.
  3. Once you have finished amending the header.inc file save the amended version and test the forum in your web browser to see if it looks and runs correctly.
  4. Open the file side_bar.inc in an HTML editor and place into this anything like navigation menus you would like displayed on the side of each page of the forum. As there is no size to the side bar by default it is recommended that you use a table to place your content into and set the width of it in pixels up to around 130pixels.
  5. Once you have finished amending the side_bar.inc file size the amended version and test the forum in your web browser to see if it looks and runs correctly.
  6. Open the file footer.inc in an HTML editor and add any content that you would like displayed at the bottom of each page of the forum.
  7. Once you have finished amending the footer.inc file size the amended version and test the forum in your web browser to see if it looks and runs correctly.
